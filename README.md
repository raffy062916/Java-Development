# Java Development: Strutture Dati e Programmazione a Oggetti

Benvenuti nel mio repository dedicato all'apprendimento di Java. In questo spazio ho raccolto gli esercizi focalizzati sull'applicazione pratica della logica di programmazione, con un forte focus sulla gestione dei dati e sulla modellazione ad oggetti.

## 📂 Struttura del Percorso

Il repository è organizzato per mostrare l'evoluzione del codice, dalle basi logiche all'architettura software:

### 1. 📊 Array e ArrayList
In questa sezione ho integrato la logica di controllo direttamente con le strutture dati:
* **Iterazioni e Controllo**: Uso di cicli (`for`, `while`) e condizioni (`if-else`, `switch`) applicati alla manipolazione di Array e ArrayList.
* **Gestione Dati**: Algoritmi per l'inserimento, la ricerca e il filtraggio di elementi all'interno di liste dinamiche.

### 2. 🏛️ Programmazione a Oggetti (OOP)
Sviluppo di applicazioni basate sui pilastri della OOP per risolvere problemi reali:
* **Modellazione**: Creazione di classi per la gestione di entità (es. Dipendenti, Ordini, Tornei).
* **Incapsulamento**: Protezione dei dati tramite modificatori di accesso e metodi Getter/Setter.
* **Logica di Business**: Implementazione di metodi specifici all'interno delle classi per la gestione dei processi.
